# Weather_Forcast
Developed a Python-based Weather Forecast desktop application using Tkinter and OpenWeatherMap API.
A simple and interactive Weather Forecast desktop application built using Python and Tkinter.
The application fetches real-time weather data using a public weather API and displays current weather conditions for any city entered by the user.
